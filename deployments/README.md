# Across Deployment Addresses

This is a legacy Hardhat deployment folder.

`legacy-addresses.json` is still kept here for older Hardhat deployments. The current source of truth for deployed contract addresses is [`broadcast/deployed-addresses.json`](../broadcast/deployed-addresses.json), which is generated by `yarn extract-addresses`.

For per-chain legacy Hardhat deployment artifacts, see the subdirectories in this folder.
