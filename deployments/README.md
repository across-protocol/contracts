# Across Deployment Addresses

The source of truth for deployed contract addresses is [`broadcast/deployed-addresses.json`](../broadcast/deployed-addresses.json), which is auto-generated by `yarn extract-addresses`.

Legacy Hardhat deployment addresses (no longer actively maintained) can be found in [`legacy-addresses.json`](./legacy-addresses.json).

For per-chain Hardhat deployment artifacts (ABIs, bytecode, etc.), see the subdirectories in this folder.
